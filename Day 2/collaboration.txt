Branch Commands --

        git branch  (to check branch)

        git branch -M main  (to rename branch)

        git checkout <branch name>  (to navigate)

        git checkout -b <new branch name>  (to create new branch)

        git branch -d <branch name>  (to delete branch)


Merging Code --

        Way 1:
            git diff <branch name>  (to compare commits, branches, files and more)

            git merge <branch name>  (to merge 2 branches)

        Way 2:
            Create a PR (Pull Request)


PR: It lets you tell others about changes you have pushed to a branch in a repository on GitHub.



Pull Command --
        git pull origin main

        used to fetch and download content from a remote repo and immediately update the local repo to match the content.
